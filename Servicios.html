Ofrecemos un sin fin de servicios, los cuales se pueden ver a continuacion:

Formateo e Instalacion de SO

Se realiza una limpieza a fondo de una computadora, desisnstalando el sistema operativo, para despues generar la instalacion de un nuevo sistema operativo, para que la computadora funcione mas rapido y sin ningun problema.

Instalacion de sowftware

Se le pide al usuario una lista de programas que nescesita que se le instalen a su computadora para poder instalarlos y entregarle la computadora con los programas funcionando con cuentas 100% seguras y originales.

Armado de computadoras

Se genera un presupuesto en conjunto con el usuario para utilizarlo y de esta manera comprar los elementos nesesarios los cuales podemros adquirir de diferentes proveedores que nos ayudan a que la computadora salga los mas economico posible.

Para poder verificar los elementos disponibles, o al menos algunos de los que se tienen en la tienda fisica, puede dirigirse a la parte superior en el aoartado "Productos".

limpieza externa de dispositivos

Se dispone a que el cliente traiga el dispositivo, o los dispositivos que desea que se limpien, para darle un trato mas fondo y mas completo, dependiendo de lo que se nescecite limpiar, sera el precio por lo que le solicitamos que antes nos mande un mensaje en la parte inferior con sus datos y con lo que desea que se le haga al equipo para poder cotizarle

Elaboracion de paginas web

Se queda a dispocicion del usuario para que en una reunion, se den todos los datos y detalles de lo que quiere que contenga la pagina a crearse, una vez hecho eso, nos pondremos a trabajar para la creacion de la misma y posterirormente realizar los detalles correspondientes para que quede como el cliente la pidido, si hubiera algun detalle que se tuviera que arreglar se vera por separado y con un costo adicional.

Acomodo de set-up

Se piden las especificaciones de el cuarto donde se va a colocar el set-up, asi como todos los dispositivos que se pondran, para plantear una idea personalizada de como quedaria, para posteriormente si la idea le gusta al cliente se realiza el proceso necesario para el cambio completo del set-up.
